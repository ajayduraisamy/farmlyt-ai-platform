# 🌱 Farmlyt AI Platform

**Empowering Agriculture Through Artificial Intelligence**

Farmlyt AI Platform is a comprehensive AI-powered agricultural intelligence system designed to provide advanced plant disease detection, healthy/unhealthy classification, plant identification, crop analysis, food grain recognition, and agricultural decision support.

Built using **Flask backend architecture**, **YOLOv8 deep learning models**, **multi-model detection systems**, **mobile applications**, and **web dashboards**, the platform delivers scalable and practical smart agriculture solutions.

---

# 🚀 Core System Overview

Farmlyt AI Platform consists of multiple AI modules trained for different agricultural categories, with each model capable of detecting:

* Healthy crops
* Unhealthy crops
* Disease categories
* Plant type classifications
* Crop-specific disease analysis
* Agricultural recommendations

# 🔍 Detection Workflow

1. User uploads image via mobile app or web dashboard
2. Flask API receives image
3. Appropriate YOLOv8 model is selected based on category
4. AI model predicts:

   * Crop type
   * Disease type
   * Healthy/unhealthy status
5. Results returned to frontend
6. User receives diagnosis and recommendations
